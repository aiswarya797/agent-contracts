# Privacy

agent-contracts is local-only by default.

- It reads files from the repository path you provide.
- It writes generated contract files only after approval.
- It does not upload source.
- It does not require a network service.
- It does not require an external API key.
- It does not install dependencies.

Generated context packs are written under the local repository by default. Treat those packs as source-bearing artifacts and store or delete them according to your team policy.
